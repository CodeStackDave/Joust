# Joust

## Requirements

- Node.js ~v5.3.0
- `npm install -g electron-prebuilt typescript tsd gulp`


## Install

Install dependencies:

```
npm install
```

Install additional Typescript typings:

```
tsd install
```

Either compile once:

```
gulp compile
```

Or recompile automatically:

```
gulp watch
```

## Run

```
electron .
```
